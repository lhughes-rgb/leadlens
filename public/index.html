WordPress Contact Form
Capture leads from any WordPress site directly into LeadLens
Active
Add this PHP snippet to your WordPress theme's functions.php file or inside a custom plugin. It hooks into Contact Form 7's submission event and forwards each lead to your LeadLens API automatically.
PHP Snippet
// LeadLens -- WordPress CF7 Integration // Add to functions.php or a custom plugin add_action( 'wpcf7_mail_sent', 'leadlens_capture_cf7_lead' ); function leadlens_capture_cf7_lead( $contact_form ) { $submission = WPCF7_Submission::get_instance(); if ( ! $submission ) return; $data = $submission->get_posted_data(); $lead = [ 'name' => sanitize_text_field( $data['your-name'] ?? '' ), 'email' => sanitize_email( $data['your-email'] ?? '' ), 'phone' => sanitize_text_field( $data['your-phone'] ?? '' ), 'source' => 'Organic', 'status' => 'New', 'client' => sanitize_text_field( $data['your-client'] ?? get_bloginfo( 'name' ) ), 'notes' => sanitize_textarea_field( $data['your-message'] ?? '' ), ]; $api_url = 'https://your-leadlens-api.com/api/leads'; wp_remote_post( $api_url, [ 'headers' => [ 'Content-Type' => 'application/json' ], 'body' => wp_json_encode( $lead ), 'timeout' => 10, ]); }
1
Install and activate Contact Form 7 on your WordPress site.
2
Replace https://your-leadlens-api.com with your actual LeadLens API URL.
3
Paste the snippet into functions.php via Appearance → Theme File Editor or a custom plugin.
4
Confirm CF7 field names match: your-name, your-email, your-phone, your-message.
5
Submit a test form and verify the lead appears in All Leads.
REST API Endpoint
Direct HTTP API for custom integrations, Zapier and Make
Active
POST JSON to create leads, PATCH to update, DELETE to remove. Pass your key in the X-API-Key header.
cURL Example
curl -X POST https://your-leadlens-api.com/api/leads -H "Content-Type: application/json" -H "X-API-Key: YOUR_API_KEY" -d '{ "name": "Jane Smith", "email": "jane@example.com", "phone": "555-0100", "client": "Apex Roofing", "source": "SEO", "status": "New", "value": 1500, "notes": "Interested in full roof replacement" }'
